# 대규모 시스템 설계 - 게시판

# **프로젝트 요구사항**

## **1. 게시글**

- 게시글 조회
- 게시글 생성
- 게시글 수정
- 게시글 삭제
- 게시글 목록 조회

## **2. 댓글**

- 댓글 조회
- 댓글 생성
- 댓글 삭제
- 댓글 목록 조회
- **계층형 구조**
    - 최대 2 depth 또는 무한 depth 지원

## **3. 게시글 좋아요**

- 게시글에 '좋아요' 기능
- '좋아요' 수 집계 및 표시

## **4. 게시글 조회수**

- 게시글 조회수 집계 및 표시
- 사용자별 10분에 1회만 집계

## **5. 인기글**

- **일일 인기글 선정**
    - 매일 상위 10개의 인기글 선정
    - 업데이트 시간: 매일 오전 1시
    - 선정 기준: 댓글 수, 좋아요 수, 조회수를 기반으로 한 점수제
- **인기글 내역 조회**
    - 최근 N일 동안의 인기글 내역 조회 기능

## **6. 최적화 전략**

- **게시글 조회 최적화**
    - 게시글 단건 조회 최적화
    - 게시글 목록 조회 최적화
- **캐시 최적화**
